RegonAPI - API Client for Polish REGON database

  • Github:  https://github.com/rolzwy7/RegonAPI

Summary

API Client for Polish REGON database (Baza Internetowa Regon - BIR). Application supports BIR version 1.0, 1.1 and is capable of searching for information about business entities by KRS/NIP/REGON numbers and Fetching business entities reports.
